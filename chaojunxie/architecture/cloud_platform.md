## 云平台

+ IaaS(infrastructure as a Service)

  + 云虚拟机

    区域内和可用区->跨区域，跨可用区部署，做高可用

    内存，cpu和存储分离

    选择纬度

    1. 系列
       + 通用均衡型
       + 计算密集型
       + 内存优化型
       + 图形计算型
    2. 代
    3. size
       + medium
       + large(2 vcpu)
       + xlarge(4 vcpu)

    叫法

    + 阿里云:ECS
    + AWS:EC2
    + Azure:Virtual machine

  + 云硬盘

    + 基于HDD->便宜
    + HDD+SSD->用SSD放热数据或者缓存
    + 纯SSD
    + 也可以使用本地硬盘，这样对大数据集群帮助比较大

  + 云上虚拟网络

  + 云数据库
